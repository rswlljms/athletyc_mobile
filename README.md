## Getting Started

### Prerequisites

- Flutter SDK (^3.6.0)
- Dart SDK
- Android Studio / VS Code
- Git

### Installation

1. Clone the repository
```bash
git clone [your-repository-url]
```

2. Navigate to the project directory
```bash
cd athletyc_mobile
```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
